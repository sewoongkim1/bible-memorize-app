# 말씀 한 절 암송 테스트

고척교회 "말씀 한 절 암송 소책자"(26주, 시편 119:105 ~ 히브리서 3:1)를
온라인에서 자가 테스트할 수 있는 웹앱입니다.

## 구성
- **프론트엔드**: Vanilla JS / HTML / CSS (GitHub Pages 배포)
- **백엔드**: Google Apps Script (Web App, `doGet`)
- **데이터**: Google Sheets

## 사용법
1. 목록에서 암송할 구절(주차)을 선택합니다.
2. 단계별 빈칸을 채워 자가 테스트합니다.
   - 1단계 25% · 2단계 65% · 3단계 100% 빈칸
   - 정답을 입력하면 자동으로 다음 빈칸으로 이동
3. 진행 상태는 브라우저에 자동 저장되어, 다음 방문 시 이어서 할 수 있습니다.

## 로컬 실행
정적 파일이므로 `index.html`을 브라우저로 열거나, Live Server 등으로 실행하면 됩니다.
